### 倾斜模块sw-520d

模块代码地址：[rvboards/d1_demo: The demo projects for Allwinner D1 SBC (github.com)](https://github.com/rvboards/d1_demo)

代码运行方法（需要sudo权限）：

```
sudo ./sw520dgpio44.sh
```

需要使用杜邦线连接模块；
